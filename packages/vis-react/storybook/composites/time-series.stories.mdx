# Time Series

This story shows how to use the Time Series composite component which consists of two area charts.
The bottom chart acts as a navigation component to the main chart. Try selection a range there and see what happens.

<Canvas>

### How to use
```tsx
// Import the component
import { VisTimeSeries } from '@volterra/vis-react/composites/time-series'

// Use in your JSX code
<VisTimeSeries
  data={data}
  x={d => d.timestamp}
  y={d => d.value}
  xLabel="Time"
  yLabel="Value"
  formatXTicks={(timestamp) => format(new Date(timestamp), 'EEE, dd MMM')}
/>
```
