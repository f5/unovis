# Line
### Basic configuration
The *Line* component has been designed to work together with *XY Container*. The minimal *Line* configuration looks like:

<LineDoc showContext={true}/>

### Multiple Lines
*Line* can also accept an array of `y` accessors to display multiple lines:
<LineDoc showContext={true} y={[d => d.y, d => d.y1, d => d.y2]}/>

### Colored Line
<LineDoc color="blue"/>

### Colored Lines
<LineDoc showContext={true} y={[d => d.y, d => d.y1, d => d.y2]} color={(d,i) => ["blue", "green", "yellow"][i]}/>
